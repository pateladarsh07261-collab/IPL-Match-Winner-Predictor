# IPL Match Winner Predictor

A Machine Learning project that predicts IPL match winners using historical IPL match data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Features
- Team Selection
- Venue Selection
- Toss Winner Selection
- Match Type Selection
- Winner Prediction
- Confidence Score

## Model
Random Forest Classifier

## Accuracy
Test Accuracy: 57.14%

## Files
- app.py → Streamlit Web App
- copyFile.ipynb → Model Training Notebook
- matches_final.csv → Dataset
- rf_pipeline.pkl → Trained Model
